⚖️ Microcontroller vs. Microprocessor — KEY EXAM TOPIC

This is the most important distinction for questions about Raspberry Pi vs Arduino/ESP32.

🤖 Microcontroller (Arduino / ESP32)

  • Entire computer on a single chip (CPU + RAM + Flash + I/O)
  • Designed for a single dedicated task
  • Very low power consumption
  • No operating system (bare metal or RTOS)
  • Limited RAM (2KB–520KB)
  • Runs at lower clock speeds (16–240 MHz)
  • Cannot run full Linux/Windows
  • Best for: sensor reading, actuator control

🍓 Microprocessor (Raspberry Pi)

  • High-performance CPU — needs external RAM, storage
  • Can run a full operating system (Linux)
  • Higher power consumption
  • Runs 32-bit and 64-bit programs
  • Large RAM (1–8 GB)
  • Clock speed: 700 MHz – 1.5 GHz
  • Can multitask like a full PC
  • Best for: ML, image processing, complex computing
🎯
The #1 Exam Answer

The main difference between Raspberry Pi and other microcontrollers: Raspberry Pi is a microprocessor / Single Board Computer (SBC) that can run a full operating system (Linux), has 1–8 GB RAM, and can perform complex tasks like ML and image processing. Arduino and ESP32 are microcontrollers that run bare-metal code for dedicated tasks.

When to Choose Each? (3-mark answer format)

Use ESP32 over Raspberry Pi when:

  • You need low cost wireless IoT solution
  • The task is simple sensor monitoring + data transmission
  • You need long battery life (low power)
  • No OS needed — just WiFi + simple logic

Use Raspberry Pi when:

  • You need complex computing (ML, computer vision)
  • Running a full server or web app
  • You need to run multiple applications
  • High processing power required

Home Security System Scenario

🏠
Microcontroller vs Microprocessor Role

Sensor Nodes → Use microcontrollers (Arduino/ESP32): low power, simple task (detect motion, read door sensor), cheap to deploy many units.

Central Monitoring Unit → Use Raspberry Pi (microprocessor): receives all data, runs web dashboard, processes video, sends alerts — needs full OS and computing power.

🍓 Raspberry Pi Overview

Raspberry Pi is a credit card-sized minicomputer designed for education and prototyping. It has a motherboard-like design but without peripherals — you connect your own keyboard, mouse, and display.

  • Compatible with Linux OS (Raspberry Pi OS)
  • Requires a microSD card for storage and OS
  • Interoperable with any input/output hardware
  • Can run both 32-bit and 64-bit programs

Raspberry Pi Model B — The Original

First iteration with ARM11 processor. Credit card-sized form factor. Also released: simpler Model A.

2014

Model B+ / A+ Enhanced Design

Improved layout, more USB ports, better GPIO. Same ARM11 cores.

February 2015

Raspberry Pi 2

900 MHz quad-core ARM Cortex-A7, 1 GB RAM. First quad-core Pi.

February 2016

Raspberry Pi 3 Model B

1.2 GHz quad-core ARM Cortex-A53. Added onboard 802.11n WiFi and Bluetooth. Game changer.

2018

Raspberry Pi 3 Model B+

1.4 GHz CPU, dual-band 802.11ac WiFi, PoE support, faster Ethernet.

June 2019

Raspberry Pi 4 Model B ⭐ (What we use in lab)

1.5 GHz quad-core ARM Cortex-A72. Full gigabit Ethernet, USB 3.0, 1–8GB RAM, dual 4K HDMI, USB-C power, BT 5.0.

📌 GPIO Pins — General Purpose Input/Output

GPIO (General Purpose Input/Output) pins are versatile interfaces for connecting the Raspberry Pi to electronic components (LEDs, motors, sensors). They allow two-way exchange of information.

⚡
GPIO Voltage Levels

GPIO pins: HIGH = 3.3V (turns component ON), LOW = 0V (turns component OFF). Unlike Arduino which uses 5V for HIGH!

GPIO Numbering

There are two ways to refer to GPIO pins:

  • GPIO Numbering (BCM/Broadcom) — Used in code. e.g., GPIO 14, GPIO 25. This is what gpiozero uses.
  • Physical Pin Number — The physical location on the 40-pin header. e.g., GPIO 25 = Physical Pin 22.

⚠️ Exam Alert: In the lab we use BCM (Broadcom) numbering in code. GPIO 14 corresponds to physical pin 8. Cannot use GPIO 0 and GPIO 1 for LED circuits.

🛠️ Setting Up Raspberry Pi

Required Hardware

  • Raspberry Pi 4 Model B board
  • USB-C power supply (minimum 3A / 5V)
  • MicroSD card (minimum 8GB)
  • USB Keyboard + USB Mouse
  • TV or monitor with HDMI + micro-HDMI cable

OS Installation Steps

1

Download Raspberry Pi Imager

Visit raspberrypi.com/software, download the Imager for your computer's OS.

2

Flash the SD Card

Insert microSD into your computer. In Imager: select OS (Raspberry Pi OS) → select SD card → click WRITE.

3

Connect Hardware

Insert SD card into Pi. Connect mouse, keyboard, HDMI to HDMI0 port (nearest to power port). Connect power last.

4

Power On & Setup

Pi boots automatically. Follow Welcome wizard: set Country, Language, Timezone, password, WiFi. Update software. Restart.

🐍 gpiozero Library

The gpiozero library is a Python library that provides easy-to-use interfaces for common components: LEDs, buttons, sensors, motors.

LED Wiring for Lab

LED (+) anode → 220Ω resistor → GPIO 14 (Pin 8) LED (-) cathode → GND (Pin 6) Note: Can use any GPIO except GPIO 0 and GPIO 1

🐍 Python Code Examples

Blink LED with gpiozero

Python 3
blinking_led.py
from gpiozero import LED from time import sleep led = LED(14) # GPIO 14 (BCM numbering) while True: led.on() # turn LED ON (3.3V HIGH) sleep(1) # wait 1 second led.off() # turn LED OFF (0V LOW) sleep(1) # wait 1 second

Using toggle() — Simpler Blink

Python 3
toggle method
from gpiozero import LED from time import sleep led = LED(14) while True: led.toggle() # reverses current state (ON→OFF or OFF→ON) sleep(1)

Button + LED (Lab Report Task 1)

Python 3
button_led.py
from gpiozero import LED, Button from signal import pause led = LED(14) # GPIO 14 button = Button(2) # GPIO 2 (with internal pull-up) button.when_pressed = led.on # LED ON when button pressed button.when_released = led.off # LED OFF when released pause() # keep script running
📦
Install gpiozero

gpiozero is pre-installed on Raspberry Pi OS. If not: python3 -m pip install gpiozero

Raspberry Pi vs Arduino — Why GPIO?

❓
Why GPIO Pins on Raspberry Pi?

GPIO pins allow the Raspberry Pi to interface with and control external electronic components (LEDs, motors, sensors). Without GPIO, the Pi would be limited to purely software tasks. GPIO enables physical world interaction — bridging software and hardware.

🧠 Quick Quiz — Test Yourself

Q1. What is the main difference between Raspberry Pi and other microcontrollers like Arduino?
Raspberry Pi is a microprocessor / SBC that runs Linux OS, has 1-8GB RAM, and can handle complex computing tasks — unlike microcontrollers. ✓
Q2. What does the GND pin on a microcontroller do?
GND is the ground reference (0V). All circuits need a common ground to complete the circuit. ✓
Q3. What is the voltage level for HIGH state on Raspberry Pi GPIO pins?
Raspberry Pi GPIO HIGH = 3.3V (unlike Arduino which uses 5V for HIGH) ✓
Q4. Which Python method in gpiozero reverses the current state of a GPIO pin?
toggle() reverses the current state — ON becomes OFF, OFF becomes ON. ✓
