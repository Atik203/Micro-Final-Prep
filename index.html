Board Comparison — Must Know!

This comparison is a guaranteed exam topic. Know it cold.

Feature 🤖 Arduino Uno 📡 ESP32 🍓 Raspberry Pi 4B
Type Microcontroller Microcontroller (SoC) Microprocessor (SBC)
CPU ATmega328P @ 16 MHz Xtensa LX6 Dual-core @ 240 MHz Cortex-A72 Quad-core @ 1.5 GHz
RAM 2 KB SRAM 520 KB SRAM 1–8 GB LPDDR4
Operating Voltage 5V (HIGH) / 0V (LOW) 3.3V (HIGH) / 0V (LOW) 3.3V (HIGH) / 0V (LOW)
WiFi ✗ No ✓ Built-in 802.11 b/g/n ✓ 802.11ac
Bluetooth ✗ No ✓ BT Classic + BLE ✓ BT 5.0
Operating System None (bare metal) None / FreeRTOS Raspberry Pi OS (Linux)
Programming C/C++ (Arduino IDE) C/C++ (Arduino IDE) Python, C/C++, any language
ADC Channels 6 (10-bit) 18 (12-bit) None built-in
UART/I2C/SPI ✓ Yes ✓ Yes (more) ✓ Yes
GPIO Pins 14 digital + 6 analog 34 programmable 40 GPIO (BCM numbering)
Storage 32 KB Flash 4 MB Flash + 1.5 MB SPIFFS MicroSD card (external)
USB USB-B (programming) Micro-USB USB-C (power) + 4×USB
Cost Very low (~$5) Low (~$10) Moderate (~$35–80)
Best For Simple sensors, beginners IoT, wireless projects Complex computing, ML, vision

Exam Strategy Tips
⏱️

10 Minutes — Manage Time

MCQs first (quick 1-mark each), then 3-mark short answers, code last.

🔢

No Math Problems

Teacher confirmed — focus on concepts, code, comparisons, not calculations.

💻

Code Questions (3-4 marks)

Know LED blink modifications, setup()/loop() structure, and Serial.begin(9600).

🆚

Comparison Questions

Arduino vs ESP32 vs RPi — cost, WiFi, processing power, OS. Learn the table above.

🔧

Hardware — Experiments 1 & 2 Only

Practice wiring MQ-2 to Arduino and DHT11 to ESP32. Know pin connections by heart.

📌

Key Voltage Levels

Arduino: HIGH=5V, LOW=0V · ESP32 & RPi GPIO: HIGH=3.3V, LOW=0V.
