RAM2 KB SRAM520 KB SRAM4–8 GB LPDDR4X
GPIO HIGH Voltage5V3.3V3.3V
WiFi Built-in✗ No✓ 802.11 b/g/n✓ 802.11ac
Bluetooth✗ No✓ BT Classic + BLE✓ BT 5.0
Operating SystemNone (bare metal)None / FreeRTOSRaspberry Pi OS (Linux)
ProgrammingC/C++ (Arduino IDE)C/C++ (Arduino IDE)Python, C/C++, any language
ADC Channels6 (10-bit)18 (12-bit)None built-in
GPIO Pins14 digital + 6 analog34 programmable40 GPIO (BCM numbering)
Storage32 KB Flash4 MB FlashMicroSD card (external)
CostVery low (~$5)Low (~$10)Moderate (~$60–80)
Best ForSimple sensors, beginnersIoT, wireless projectsComplex computing, ML, vision

Exam Strategy Tips
⏱️

20 Minutes — Manage Time

MCQs first (quick 1-mark each), then 3-mark short answers, code modification last.

🔢

No Math Problems

Teacher confirmed — focus on concepts, code, comparisons, not calculations.

💻

Code Questions (3–4 marks)

Know LED blink modifications, setup()/loop() structure, and Serial.begin(9600).

🆚

Comparison Questions

Arduino vs ESP32 vs RPi — cost, WiFi, processing power, OS. Learn the table above.

🔧

Hardware — Ex 1 & 2 Only

Practice wiring MQ-2 to Arduino and DHT11 to ESP32. Know pin connections by heart.

📌

Key Voltage Levels

Arduino: HIGH=5V, LOW=0V · ESP32 & RPi GPIO: HIGH=3.3V, LOW=0V.
