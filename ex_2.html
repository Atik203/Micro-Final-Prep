📡 ESP32 Overview

The ESP32 is a widely-used System on a Chip (SoC) that plays a key role in developing Internet of Things (IoT) devices. It combines WiFi, Bluetooth, dual processors, and rich peripherals in one chip.

🌐
Key Distinction

ESP32 is the board to choose when you need wireless IoT connectivity at low cost. Arduino has no WiFi/BT; Raspberry Pi is powerful but expensive. ESP32 is the sweet spot for IoT.

⚡ Key Features of ESP32

📶 Built-in WiFi + BT

802.11 b/g/n WiFi and Bluetooth Classic + BLE. No external module needed — ideal for IoT.

🔒 Security

Hardware encryption support ensures secure data communication — critical for IoT deployments.

⚙️ Dual Core

Two Xtensa LX6 cores at 240 MHz. Enables multitasking and fast parallel processing for real-time apps.

💾 Storage

1.5 MB SPIFFS storage for files on external flash. 4 MB total flash. Useful for web files, configs.

🧠 RAM

520 KB SRAM — handles data and operations efficiently for IoT applications.

🔌 Sensor Compatible

Supports UART, I2C, SPI, ADC, capacitive touch. Interfaces with a wide range of IoT sensors.

3 Key Features for IoT (Exam Answer): 1) Built-in WiFi & Bluetooth connectivity, 2) Dual-core processor for real-time tasks, 3) Hardware encryption for secure communication.

Important Note

ADC and DAC pins are fixed (static). However, UART, I2C, SPI, and PWM can be reassigned to any GPIO by code — this makes ESP32 very flexible.

🌡️ DHT11 Sensor

The DHT11 measures temperature and humidity. In this experiment, it's connected to the ESP32 at pin D19.

==== DHT11 to ESP32 Connection ==== DHT11 VCC → ESP32 3.3V (or 5V via regulator) DHT11 GND → ESP32 GND DHT11 DATA → ESP32 D19 (#define DHTPIN 19) Note: Add 10K pull-up resistor between DATA and VCC

DHT11 Code for ESP32

⬡ Arduino C++
DHT11 with ESP32
#include <DHT.h> #define DHTPIN 19 // GPIO 19 = D19 #define DHTTYPE DHT11 DHT dht11(DHTPIN, DHTTYPE); // create DHT object int temperature = 0; int humidity = 0; void setup() { Serial.begin(115200); // ESP32 uses 115200 baud dht11.begin(); // initialize sensor } void loop() { temperature = dht11.readTemperature(); // Celsius humidity = dht11.readHumidity(); if (isnan(temperature) || isnan(humidity)) { Serial.println("Sensor read failed!"); return; } Serial.printf("Temp: %d C, Humidity: %d%%\n", temperature, humidity); }
⚠️
Upload Tip

If code upload fails on ESP32: Hold the BOOT button + press the EN button, then click Upload in the IDE. Release BOOT after upload starts.

🖥️ XAMPP Overview

XAMPP is a free, open-source local web server that lets developers create and test websites on their own computer. It runs a mini internet server locally. The name is an acronym:

📘
XAMPP Acronym

X = Cross-Platform  |  A = Apache  |  M = MySQL  |  P = PHP  |  P = Perl

How They Work Together

Apache
Web server — serves pages
⟷
PHP
Processes data, creates dynamic pages
⟷
MySQL
Stores & manages data

🔧 Apache, MySQL, and PHP

What it does: Takes requests from web browsers and delivers the requested web pages.

Example: When you type "www.example.com", Apache delivers the site content to your browser.

In XAMPP: Serves pages at localhost. Must be started in XAMPP Control Panel.

What it does: Stores and manages data for websites.

Example: For an online store, MySQL stores product details, customer info, and order history.

In our lab: Stores temperature, humidity, and gas sensor readings from ESP32. Managed via phpMyAdmin.

SQL

What it does: A scripting language for creating dynamic web pages, runs on the server.

Example: When you log in, PHP checks username/password against MySQL and directs you to your account.

In our lab: Receives data POSTed by ESP32 and inserts it into the MySQL database.

🔧 ESP32 WiFi + HTTP Code

The ESP32 connects to WiFi and sends sensor data to the XAMPP server using HTTP POST requests.

⬡ Arduino C++
ESP32 → XAMPP Server
#include <WiFi.h> #include <HTTPClient.h> #include <DHT.h> #define DHTPIN 19 #define DHTTYPE DHT11 DHT dht11(DHTPIN, DHTTYPE); // Replace with your computer's IP (ipconfig command) String URL = "http://192.168.1.X/dht11_project/testdata.php"; const char* ssid = "YOUR_WIFI_SSID"; const char* password = "YOUR_WIFI_PASSWORD"; int temperature = 0, humidity = 0; void setup() { Serial.begin(115200); dht11.begin(); connectWiFi(); } void loop() { if (WiFi.status() != WL_CONNECTED) connectWiFi(); temperature = dht11.readTemperature(); humidity = dht11.readHumidity(); String postData = "temperature=" + String(temperature) + "&humidity=" + String(humidity); HTTPClient http; http.begin(URL); http.addHeader("Content-Type", "application/x-www-form-urlencoded"); int httpCode = http.POST(postData); // 200 = success String payload = http.getString(); Serial.println(httpCode); delay(5000); // send every 5 seconds } void connectWiFi() { WiFi.mode(WIFI_OFF); delay(1000); WiFi.mode(WIFI_STA); WiFi.begin(ssid, password); Serial.print("Connecting..."); while (WiFi.status() != WL_CONNECTED) { delay(500); Serial.print("."); } Serial.println(WiFi.localIP()); }

Key Code Concepts

  • WiFi.begin(ssid, password) — connects to WiFi network
  • HTTPClient http — creates HTTP client object
  • http.begin(URL) — sets the target server URL
  • http.POST(postData) — sends data via HTTP POST; returns 200 on success
  • http.getString() — gets the server response (e.g., "New record created")
  • Use ipconfig in Windows CMD to find your computer's IP address

🔌 Complete Wiring Summary

==== DHT11 to ESP32 ==== DHT11 VCC → ESP32 3.3V DHT11 GND → ESP32 GND DHT11 DATA → ESP32 GPIO19 (D19) ==== MQ-2 Gas Sensor to ESP32 ==== MQ-2 VCC → ESP32 5V (or VIN) MQ-2 GND → ESP32 GND MQ-2 AO → ESP32 GPIO34 (ADC1) MQ-2 DO → ESP32 any GPIO ==== LED to ESP32 ==== LED (+) → ESP32 GPIO (via resistor) LED (-) → ESP32 GND

⚙️ XAMPP Setup Steps

1

Download & Install XAMPP

Get from apachefriends.org. Run the installer — straightforward process.

2

Start Apache & MySQL

Open C:\xampp\xampp-control.exe. Click "Start" for both Apache and MySQL modules.

3

Verify server works

Type localhost in browser. You should see XAMPP welcome page.

4

Create Database via phpMyAdmin

Go to localhost/phpmyadmin. Create new database (e.g., "sensor_db"), then create a table with columns: ID, temperature, humidity, Datetime.

5

Create PHP file

Create project folder in C:\xampp\htdocs\. Create testdata.php with connection and INSERT code.

6

Upload ESP32 code

Set WiFi credentials and server URL (use ipconfig to get IP). Upload and monitor serial output for "200" response code.
